Git Commands task

    Commands:

  1. git clone - Clone a Repo
  2. git add - Stage a file
  3. git commit - Confirm to commit changes on a local repo
  4. git push - Push a changes into remote

    Git branch commands

  1. git checkout -b "branch1" - create a new branch and switch to that
  2. git branch - check all branches and current branch where you're in
  3. git pull origin "branch1" - sync the remote to local
  4. git merge - merge branches
